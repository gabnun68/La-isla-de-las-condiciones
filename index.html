📘 Aprende sobre condicionales

Un condicional en Python permite ejecutar ciertas instrucciones solo si se cumple una condición:

if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres niño/a")
